动态转换 dynamic_cast<float>（2）;将基类指针转换为派生类指针
常量转换：
const int i = 10;
int& r = const_cast<int&>(i); // 常量转换，将const int转换为int 只去除const
重新解释转换：int i = 10; float f = reinterpret_cast<float&>(i); // 重新解释将int类型转换为float类型
volatile	修饰符 volatile 告诉该变量的值可能会被程序以外的因素改变，如硬件或其他线程
mutable	mutable 用于修饰类的成员变量。被 mutable 修饰的成员变量可以被修改，即使它们所在的对象是 const 的。
static	用于定义静态变量，表示该变量的作用域仅限于当前文件或当前函数内，不会被其他文件或函数访问。
